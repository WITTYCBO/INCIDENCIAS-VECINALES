Instrucciones de despliegue

  1. Publica el contenido de esta carpeta (web/) en un alojamiento accesible o inserta el HTML directamente en Google Sites.
  2. Crea un proyecto de Google Apps Script conectado a una hoja de cálculo de Google Sheets.
  3. Actualiza la constante APPS_SCRIPT_URL del archivo app.js con la URL del Web App desplegado.
  4. Publica el Web App con acceso a "Cualquiera con el enlace" para permitir el envío del formulario.
